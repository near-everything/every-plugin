## Key Design Decisions

1. **Node.js for development** - Full hot reload and debugging
2. **WASI for production** - Secure, multi-tenant, fast cold starts
3. **Documentation in fumadocs** - Static site generation, MDX support
4. **Module Federation** - Dynamic UI loading without npm installs
5. **every-plugin for APIs** - Type-safe, contract-first plugin system
